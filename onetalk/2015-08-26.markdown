# 不是丰饶，而是通缩

![QR](qrcode.jpg)

MM: 今年开始身边越来越多的人开始各种健身啊、减肥啊什么的。朋友圈里一下子冒出很多晒跑步的朋友。

EL: 你想过没有，健身潮的兴起和互联网电商的繁荣、经济的衰退、股灾这些形形色色的事情之间有什么样的联系？

MM: 有怎样的联系呢？

EL: 与互联网商业有关的各种理论中，有一个概念叫做丰饶经济。这个概念的意思是说，随着工业文明的进步，生产力的不断提高，人们已经可以生产出远远超出需要的商品，因此，在这种情况下，商品供给极大丰富，这便叫做丰饶经济。

MM: 那么丰饶经济和电商繁荣有何关系呢？

EL: 真正最大程度上反映了丰饶经济时代特点的不是像京东那样的自营型电商，而是淘宝那样的平台型电商。淘宝本质上是一个商品广告平台和流量贩子。丰饶经济主导下的商品零售市场，供给的极大丰富反衬出了需求的相对稀缺。而淘宝正是从中汇聚、收购这些相对稀缺的需求，形成流量，然后把这些流量精准地贩卖给后端的成千上百万的中小卖家们。

然后淘宝就可以躺着向中小卖家们收取贩卖流量的中介服务费了。

阿里巴巴上市于2014年9月。这是一个互联网电商达至鼎盛的标志性事件。

MM: 供给极大丰饶，这不是人类一直向往的美好景象吗？

EL: 但是你有没有进一步想过，丰饶经济这个词或者只是对一个非常可怕的经济状况的美好包装？
供给丰饶的情状也许只是具备强大的媒体属性的互联网企业和畅销书作者们共同编织的美好幻境？

MM: 那么丰饶经济的同义词是什么呢？

EL: 产能过剩。这些年来，我们，以及我们的政府，一直相搏斗的对象。

产能过剩的后继，是失业，是经济增长的停滞，是衰退，是股灾，是通货紧缩。

股灾于2015年8月。这就是所谓丰饶经济华丽包装背后赤裸裸的真相。

MM: 股灾的原因不是政府去金融杠杆吗？

EL: 政府去杠杆只是一个导火索。关键是杠杆已经把银行和风险捆绑。去杠杆更多是一种被迫的自卫行为。

我们只有金融有杠杆风险吗？我们的环境杠杆风险不是更高吗？
金融去杠杆，不过是钱没有了。而环境去杠杆，那搭进去的可都是人命啊！

刚刚发生的天津港危化物品爆炸，不就是我们过去的不规范发展所积累的环境风险所遭遇的一次去杠杆吗？

MM: 那么这一切，和健身潮的兴起有什么关系呢？

EL: 健身，本质上是人类所特有的一种高度自律的行为。健身潮兴起的现象，说明了人们开始从关注外在的欲望世界转为更加关注自身，开始节制饮食和不良生活习惯，也就是开始培养自律、节制欲望。

节制的浪潮在社会中兴起，人们不再像过去那样追求外物和享乐，也就意味着需求的持续减少。

供给的极大丰饶，不过是需求极大减少的相对效应而已。

MM: 那么未来会怎样？

EL: 当人们重新开始追逐享乐，当丰饶经济理论过气，平台型电商不再繁荣，而自营型电商成为主流，我们的经济重新回归健康。

也许彼时，我们还会坚持健身，不过那只是因为我们已经老去。

那个时代，是又一个消费需求旺盛的美丽新时代。
